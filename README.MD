# Города Трудовой Доблести (1941-1945)

## О проекте

**Города трудовой доблести** — это учебный исторический веб-проект, посвящённый городам, внёсшим значительный вклад в победу в Великой Отечественной войне.

Проект рассказывает о роли тыловых городов, их промышленном, военном и социальном вкладе, а также помогает пользователям изучать исторические материалы в удобном интерактивном формате.

---

## Функциональность

- Просмотр списка городов трудовой доблести
- Отдельная страница для каждого города
- Краткое и подробное описание исторического вклада города
- Интерактивная карта с городами
- Административная панель для добавления и редактирования материалов
- Авторизация администратора

### Инструкция по запуску:

Необходимые библиотеки для ```backend``` находятся
в файле ```requirements.txt```

```frontend:```

```
cd frontend
nmp start
```

```backend```
```
cd backend
python app.py
```
