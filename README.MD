# Города Трудовой Доблести (1941-1945)

### Описание:
Сайт по истории с городами Трудовой Доблести. На сайте есть супер админ, 
который может назначать других админов. Вся модерация на сайте происходит ими. 
Обычный пользователь может просматривать информацию о городах без регистрации. 
Также есть карта на которой отображены города Трудовой Доблести и
даны ссылки на их страницы.
---
### Инструкция по запуску:
Необходимые библиотеки для ```backend``` находятся
в файле ```requirements.txt```

```frontend:```

```
cd frontend
nmp start
```

```backend```
```
cd backend
python app.py
```
